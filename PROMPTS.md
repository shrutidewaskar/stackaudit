# StackAudit AI Prompts

This document contains the AI prompts and LLM integration guidelines for StackAudit.

## Executive Summary Generation

**Status:** Sprint 5 (Planned)  
**Purpose:** Generate a compelling 100-word executive summary of an audit result  
**Location:** `services/anthropic.ts` (to be created)

### Current Implementation

Sprint 4 uses a template-based summary generated by `services/generateSummary.ts`. This provides a professional, consistent summary while maintaining full functionality without external dependencies.

### Future AI Integration

In a future sprint, we will integrate:

- **Model:** Claude 3.5 Sonnet  
- **Service:** Anthropic API
- **Location:** `services/anthropic.ts` with fallback to template

The system will attempt to call Anthropic to generate a personalized summary, and automatically fall back to the template if the API is unavailable or if `ANTHROPIC_API_KEY` is not configured.

### Prompt (Planned)

```
You are an expert at summarizing SaaS cost optimization audits. 

Generate a compelling, professional 100-word executive summary for a company's AI tools audit report. Keep it concise and impactful.

Input data:
- Team Size: {team_size}
- Primary Use Case: {use_case}
- Current Monthly Spend: ${current_monthly_spend}
- Optimized Monthly Spend: ${optimized_monthly_spend}
- Monthly Savings: ${monthly_savings}
- Annual Savings: ${annual_savings}
- Percentage Saved: {percentage_saved}%
- Biggest Savings Opportunity: {highest_tool} (${highest_amount}/month)

Requirements:
- Approximately 100 words
- Professional, confident tone
- Highlight the key financial impact
- Emphasize the ease of implementation
- Make it compelling for a VP or Founder

Generate only the summary text, nothing else.
```
